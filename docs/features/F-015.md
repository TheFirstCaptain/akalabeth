# F-015: Modern App UX Layer

## Summary

Add modern Mac app ergonomics around the original game without hiding or replacing the historical play flow.

## Reference

- Docs: `docs/features/F-010.md`, `docs/features/F-011.md`, `BOUNDARIES.md`
- App shell: `Sources/AkalabethApp/main.swift`

## Scope

- Add native Help/About content for controls, save behavior, and compatibility notes.
- Improve restart/reset/save status feedback.
- Add display preferences such as original scale, integer scaling, high contrast, and optional scanline/CRT treatment if appropriate.
- Distinguish original behavior from modern convenience behavior in UI text and docs.

## Acceptance Criteria

- A new user can discover controls and save/reset behavior from the app.
- Settings persist and are covered by tests where practical.
- Original-mode gameplay remains available without requiring modern shortcuts.

## Validation

```bash
swift build
swift test
.build/arm64-apple-macosx/debug/AkalabethApp --smoke-test
```
