## Summary

Extract attack rules, weapon handling, magic amulet effects, monster damage, death handling, Lord British quests, and win condition into portable tested code.

## Scope

- Implement combat command resolution and damage/status events.
- Implement magic amulet outcomes and any class or inventory restrictions found in the listing.
- Implement death state, quest assignment, quest completion, and victory transitions.

## Acceptance Criteria

- Harness tests cover hit, miss, kill, damage, magic, death, quest assignment, quest completion, and final victory paths.
- Quest target names and monster ordering match the source data tables.
- All random combat outcomes can be driven by deterministic fixtures.
- Intentional modernization differences, if any, are recorded in the feature file before implementation is marked complete.
