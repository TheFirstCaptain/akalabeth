# F-017: Save Format Hardening

## Summary

Replace raw same-version memory snapshots with a versioned, portable save format suitable for real users.

## Reference

- Docs: `docs/features/F-011.md`, `BOUNDARIES.md`
- Code: `Sources/AkalabethMac/AkalabethPersistence.swift`, `Sources/AkalabethMac/GameSession.swift`

## Scope

- Define a versioned save schema for game state and adapter metadata.
- Reject or migrate incompatible saves safely.
- Add metadata such as timestamp, app version, game mode, and display name.
- Keep filesystem behavior in the Mac persistence adapter.

## Acceptance Criteria

- Save files are not raw `AkGameState` memory dumps.
- Corrupt and unsupported-version saves fail gracefully.
- Round-trip tests cover representative modes.
- Save/resume remains documented as a modern addition, not original behavior.
