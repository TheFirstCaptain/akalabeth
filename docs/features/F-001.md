# F-001: Source Listing Characterization Harness

## Summary

Create a modern command-line harness that can validate structural facts about the Akalabeth BASIC listings before behavior extraction begins.

## Reference

- `AKLABETH.TXT`
- `AKLABETH-org.TXT`
- `Core/include/ak_basic_listing.h`
- `harness/basic_listing_tests.c`

## Acceptance Criteria

- The harness parses line-numbered BASIC listings without modifying them.
- Tests verify key line counts, line ranges, and known data tables.
- `make -C harness test` passes on a modern compiler.

## Status

Implemented as the first modernization scaffold.
