# F-004: Applesoft-Compatible Random Adapter

## Summary

Implement a deterministic random-number adapter that preserves the Applesoft `RND` behaviors needed by Akalabeth, including seeded world and character generation.

## Reference

- BASIC lines: `0-99`, `500-590`, `2000-2090`, `60000-60250`
- Screenshots: `STARTUP1.GIF`, `STARTUP2.GIF`, `STARTUP3.GIF`
- Docs: `docs/modernization/legacy-inventory.md`, `docs/modernization/validation-strategy.md`

## Scope

- Characterize every source use of `RND`, including negative seeding and integer range expressions.
- Add a portable RNG interface for `Core/` and at least one deterministic implementation for tests.
- Keep Apple II-specific compatibility details behind the RNG adapter rather than spreading them through game rules.

## Acceptance Criteria

- Harness tests cover the extracted `RND` call sites and expected deterministic sequences for fixed seeds.
- Game-rule code consumes randomness through an explicit adapter or fixture.
- Open questions about exact Applesoft numeric behavior are documented when they cannot yet be proven.

## Validation

```bash
make -C harness test
```
